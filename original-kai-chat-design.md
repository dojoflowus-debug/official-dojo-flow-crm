# Original Kai Chat Page Design Analysis

## Overall Layout
- **Three-column layout**: Left sidebar (dark), Command Center sidebar (light gray), Main conversation panel (white/light)
- **Dark theme sidebar** on far left with DojoFlow logo and navigation
- **Light gray Command Center** in middle with search, tabs, collections, conversations
- **White conversation panel** on right with messages and input

## Left Sidebar (Dark - #1a1a2e style)
- DojoFlow logo at top (red/coral icon with white text)
- Navigation items with icons:
  - Kai Command (highlighted in coral/red - active state)
  - Students
  - Statistics
  - Leads
  - Classes
  - Kiosk
  - Receptionist
  - Staff
  - Billing
  - Reports
  - Marketing
  - Settings (expandable)
- User profile at bottom with avatar, name, email

## Top Bar
- "Dashboard" title on left
- Right side: "Sidebar" toggle, "Top Menu" toggle, "Credits: 0", notification icons

## Command Center Sidebar (Light Gray)
- Header: "Command Center" with icons (checkbox, +, "New Chat" button in coral)
- Search bar: "Search history, tags, @mentions..."
- Tabs: Active | Archived | All (pill style, Active highlighted)
- SMART COLLECTIONS section:
  - Urgent (1) - with clock icon
  - Kai Insights (5) - with sparkle icon
  - Pending Tasks (14) - with checkbox icon
- RECENT CONVERSATIONS section:
  - Grouped by TODAY, YESTERDAY
  - Each conversation card shows:
    - Title (truncated)
    - Timestamp
    - Preview text
    - Tags (kai, growth, billing) with "neutral" status
    - "In Progress" indicator
  - Dropdown menu button on each card

- Action buttons in top-right: Summarize, Invite Team, Voice Replies, Focus Mode

  - Textarea: "Message Kai... (Type @ to mention)"

## Color Scheme
- Primary accent: Coral/Red (#E85A5A or similar)
- Dark sidebar: Dark navy/charcoal (#1a1a2e or #0f172a)
- Command Center: Light gray (#F5F7FA)
- Conversation panel: White (#FFFFFF)
- Text: Dark gray/charcoal for readability
- Tags: Light colored pills (kai=light blue, growth=light green, billing=light purple)

## Typography
- Clean sans-serif font (likely Inter or similar)
- Navigation items: Medium weight
- Headers: Semi-bold
- Body text: Regular weight

## Key Interactions
- Conversation cards are clickable
- Each card has a dropdown menu
- Tabs switch between Active/Archived/All
- Smart collections are clickable filters
- Input bar has attachment, voice, and send buttons
