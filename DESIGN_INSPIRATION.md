## ü•ã DojoFlow Design Strategy

### Color Palette (Martial Arts + Premium SaaS)
- **Primary**: DojoFlow Red (#ED393D) - energy, passion, action
- **Secondary**: Deep Charcoal (#0F1115) - strength, discipline
- **Accent**: Gold/Amber (#F59E0B) - achievement, excellence
- **Gradients**: Red ‚Üí Orange for warmth, Charcoal ‚Üí Purple for depth
- **Background**: Clean white (light mode) or deep black (dark mode)

### Typography System
- **Display**: Bold, confident sans-serif (Inter Black or Satoshi Black)
  - Hero: 72-96px
  - Section Headers: 48-60px
- **Headings**: Inter Bold or Satoshi Bold (24-36px)
- **Body**: Inter Regular (16-18px, line-height 1.6)
- **Code/Data**: JetBrains Mono for technical elements

### Layout Architecture
1. **Hero Section**
   - Full-screen with animated gradient background
   - Massive headline: "Run Your Dojo Like a Sensei"
   - Subheadline: AI-powered management for martial arts schools
   - Dual CTAs: "Start Free Trial" + "Watch Demo"
   - Animated dashboard preview or 3D dojo visualization

2. **Problem/Solution Section**
   - "We understand your pain" (inspired by Pandawa)
   - 3 pain points with icons
   - Transition to solution with visual

3. **Feature Showcase (Bento Grid)**
   - Mixed-size cards with screenshots
   - Hover effects reveal more info
   - Icons + short descriptions
   - Key features: Kai AI, Student Tracking, Billing, Kiosk

4. **Comparison Section**
   - "Why DojoFlow?" side-by-side table
   - DojoFlow vs Traditional Software vs Spreadsheets
   - Clear visual differentiation

5. **Kai AI Spotlight**
   - Dedicated section for AI assistant
   - Interactive demo or video
   - "Your AI Sensei" messaging

6. **Social Proof**
   - Testimonials from dojo owners
   - Stats: "X schools managed, Y students tracked"
   - Logo wall of martial arts associations

7. **Pricing Section**
   - 3-4 tiers with clear differentiation
   - "Most Popular" badge
   - Feature comparison table
   - Monthly/Annual toggle

8. **Founder Story** (inspired by Pandawa)
   - Personal connection to martial arts
   - Why we built DojoFlow
   - Photo + authentic story

9. **Final CTA**
   - "Ready to transform your dojo?"
   - Large CTA button
   - Trust signals (no credit card, 14-day trial)

10. **Footer**
    - Multi-column sitemap
    - Newsletter signup
    - Social links

### Animation Principles
- **Timing**: 300-500ms for most transitions
- **Easing**: ease-out for entrances, ease-in-out for hovers
- **Scroll Reveal**: Fade + slide up (20-40px)
- **Parallax**: Subtle depth on hero and feature images
- **Hover**: Scale (1.02-1.05) + shadow increase
- **Gradient Animation**: Slow, continuous gradient shift
- **Loading States**: Skeleton screens, not spinners

### Visual Effects Toolkit
- **Glassmorphism**: Frosted glass cards with backdrop-blur-xl
- **Grain Texture**: Subtle noise overlay (5% opacity)
- **Glow Effects**: Soft box-shadow with brand colors
- **Gradient Meshes**: Multi-stop gradients (3-4 colors)
- **3D Depth**: Layered shadows (multiple box-shadows)
- **Animated Gradients**: CSS gradient animation or canvas

### Micro-Interactions
- Button hover: scale + glow
- Card hover: lift + shadow increase
- Link hover: underline slide-in
- Input focus: border glow + scale
- Scroll progress indicator
- Cursor trail (optional, subtle)

### Mobile-First Considerations
- Stack all sections vertically
- Reduce font sizes (Hero: 48px mobile)
- Touch-friendly buttons (min 44px height)
- Simplified animations (reduce motion)
- Faster load times (lazy loading images)

## Technical Stack

- **Framework**: React 19 + Tailwind CSS 4
- **Animations**: Framer Motion for complex animations
- **Scroll**: react-intersection-observer for scroll triggers
- **Gradients**: CSS gradients + canvas for advanced effects
- **Icons**: Lucide React
- **Images**: Optimized WebP with lazy loading
- **Performance**: Code splitting, lazy loading, image optimization
