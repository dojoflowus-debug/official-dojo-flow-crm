# Pandawa Hero Banner Design Analysis

## Overall Layout Structure
- Full-screen hero section with centered content
- Large, bold headline split across multiple lines
- Prominent CTA button below headline
- Clean, minimal design with focus on typography

## Typography Patterns
- **Headline**: Very large display text, split into 2-3 lines
  - "Write" on first line
  - "with AI" on second line
  - Creates visual rhythm and emphasis
- **Subheadline**: Smaller descriptive text explaining the value proposition
- **CTA**: Clear call-to-action with trial offer

## Visual Effects & Styling
- Gradient backgrounds (likely animated)
- Glassmorphism/frosted glass effects
- Soft shadows and depth
- Premium color palette
- Smooth animations on scroll

## Spacing & Hierarchy
- Generous whitespace around headline
- Clear visual hierarchy: Headline → Subheadline → CTA
- Centered alignment for all elements
- Breathing room between sections

## Key Design Elements to Adapt for DojoFlow
1. **Split headline approach** - Break "Run Your Dojo Like a Sensei" into multiple lines for impact
2. **Centered layout** - All hero content centered vertically and horizontally
3. **Minimal distractions** - Focus on the message, remove competing elements
4. **Premium effects** - Maintain glassmorphism and gradient backgrounds
5. **Clear CTA hierarchy** - Primary action button prominently displayed
6. **Subheadline clarity** - Brief, benefit-focused description

## DojoFlow Adaptation Plan
- Keep DojoFlow red/gold branding
- Use split headline: "Run Your Dojo" / "Like a Sensei"
- Center all content
- Simplify hero to focus on message
- Add animated gradient background
- Prominent "Start Free Trial" CTA
- Brief subheadline about AI-powered management
