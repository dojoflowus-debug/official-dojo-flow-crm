# Kai Chat Original Color Reference

## Extracted from https://dojomanage-dbet6f4d.manus.space/kai-chat

### Sidebar (Dark)
- Background: #1E293B (dark slate blue)
- Logo text "DojoFlow": White
- Logo swirl: Coral/salmon (#E85A6B or similar)
- Active nav item (Kai Command): Coral/salmon background (#E85A6B)
- Inactive nav items: Light gray text on dark
- Drag handles (⋮⋮): Light gray
- User profile section: Slightly lighter dark background

### Top Bar
- Background: Same dark as sidebar (#1E293B)
- "Dashboard" text: White
- "Sidebar" button: Coral/salmon background when active
- "Top Menu" button: Gray/muted when inactive
- "Credits: 0": Green (#10B981 emerald)
- Icons: Light gray/white

### Command Center (Middle Column)
- Background: Light gray (#F5F7FB or #F8FAFC)
- "Command Center" header: Dark text
- "New Chat" button: Coral/salmon (#E85A6B) with white text
- Search input: White background with light border
- Tabs (Active/Archived/All): 
  - Active tab: White background, dark text
  - Inactive tabs: Transparent, gray text
- Smart Collections:
  - Urgent icon: Red/coral
  - Kai Insights icon: Purple/violet
  - Pending Tasks icon: Teal/cyan
  - Count badges: Light gray background

### Conversation Cards
- Background: White
- Title: Dark text
- Time: Light gray
- Tags:
  - "kai": Light blue background (#DBEAFE), blue text
  - "growth": Light green background (#D1FAE5), green text
  - "billing": Light purple background (#EDE9FE), purple text
  - "neutral": Light gray background
- Status "In Progress": Light gray text
- Three-dot menu: Light gray
- Archive button: Light gray

### Main Panel (Right)
- Background: White
- Top banner: Very light gray background
- Banner text: Small, uppercase, gray
- Action buttons: Light gray icons
- Kai logo: Coral/salmon swirl
- "Hi, I'm Kai." text: Dark, large
- Subtitle: Gray text

### Suggested Prompt Cards
- Background: White
- Border: Light gray, subtle
- Header text: Coral/salmon (#E85A6B), uppercase, small
- Body text: Dark gray, quoted

### Input Bar
- Background: Light gray (#F5F7FB)
- Border: Coral/salmon accent
- Placeholder text: Gray
- Icons: Gray
- Send button: Coral/salmon background
- Settings gear: Gray, bottom left

### Key Color Palette
- Primary Coral/Salmon: #E85A6B or #EF4444 (red-500)
- Dark Sidebar: #1E293B (slate-800)
- Light Background: #F5F7FB or #F8FAFC (slate-50)
- White: #FFFFFF
- Green (Credits): #10B981 (emerald-500)
- Tag Blue: #3B82F6 (blue-500) on #DBEAFE (blue-100)
- Tag Green: #10B981 (emerald-500) on #D1FAE5 (emerald-100)
- Tag Purple: #8B5CF6 (violet-500) on #EDE9FE (violet-100)
- Text Dark: #1E293B (slate-800)
- Text Gray: #64748B (slate-500)
